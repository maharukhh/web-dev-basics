# Web Dev Basics

A collection of frontend web development projects built with HTML, CSS, and
JavaScript, focused on core concepts like DOM manipulation, event handling,
responsive UI design, and interactive component building.

## Overview

This repository demonstrates practical frontend skills through a series of
self-contained projects — from UI components and utility tools to small
interactive applications and API-driven mini-apps. Each project is built
without external frameworks, emphasizing a strong grasp of vanilla
JavaScript and core web fundamentals before scaling into frameworks or
full-stack development.
